1. Create virtual environment: python -m venv YOUR_VIRTUAL_ENVIRONMENT_NAME
2. Activate the virtual environment: source YOUR_VIRTUAL_ENVIRONMENT_NAME/Scripts/activate
3. Install Django: pip install Django
4. Install Django Rest framework: pip install Djangorestframework
5. Create project: django-admin startproject YOUR_PROJECT_NAME .
6. Create an application: django-admin startapp YOUR_APP_NAME
7. Run application: python manage.py runserver
8. To migrate: python manage.py migrate
9. To create migrations: python manage.py makemigrations
10. To create super user: python manage.py createsuperuser
